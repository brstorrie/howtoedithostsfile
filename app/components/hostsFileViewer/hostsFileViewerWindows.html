Windows

  1. Open the Windows Start Menu, and type "Notepad"
  2. Right click Notepad, and select "Run as Administrator"
  3. Once open, select File > Open from the menu bar
  4. Once in the Open dialog, click in the address bar, input '%WINDIR%/system32/drivers/etc/', and press enter
  5. Change the "File Types" dropdown to be "All Files", and select the file named "hosts"
  6. Once open, you'll see the default hosts file information at the top; Move to a blank line, and enter the source and desired alias, separated by a space. (Usually an IP address, followed by a url-style alias, as shown below)
  7. Save the file, and open the new alias in a browser, or use command prompt to ping the alias Note: You may have to reopen any open browsers to see the change
